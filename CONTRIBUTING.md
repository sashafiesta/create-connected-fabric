# Contributing to Create: Connected (Fabric)

This is the Fabric port of [Create: Connected](https://github.com/hlysine/create_connected).

## Where to contribute

All major contributions, bug fixes, feature requests and new features should go to the
**upstream Forge repository**: https://github.com/hlysine/create_connected

This Fabric port repository only accepts:

- **Fabric-specific bug fixes** (e.g. crashes or behavior differences caused by the port)
- **Consistency updates** to keep the port in sync with upstream Forge releases
- **Dependency version bumps** for Fabric-specific libraries

## Issues

### Reporting bugs

- If the bug also happens on Forge, report it upstream: https://github.com/hlysine/create_connected/issues
- If the bug is Fabric-specific, report it here. Please include:
    - Your crash log pasted to https://mclo.gs/
    - Steps to reproduce
    - Screenshots if applicable

## Translations

Translations are managed upstream via [Crowdin](https://crowdin.com/project/create-connected-mod). They are synced to this port periodically.

## Pull requests

Before opening a PR, please check if the change applies to Forge as well. If it does, submit it upstream instead.
